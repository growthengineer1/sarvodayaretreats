# Sarvodaya Retreats

A beautiful, artistic website for a backwater retreat sanctuary near Varkala, Kerala.

## Overview

Sarvodaya Retreats is a single-page website featuring hand-drawn illustrations, organic layouts, and a peaceful aesthetic inspired by The North (thenorth.in). The site showcases an intimate retreat space designed for artists, solo travelers, spiritual practitioners, and nature lovers.

## Features

- **Artistic Design**: Hand-drawn SVG illustrations with sage green color palette
- **Fully Responsive**: Works seamlessly on mobile, tablet, and desktop
- **Interactive Elements**: Smooth scrolling, hover effects, and fade-in animations
- **Contact Form**: Built-in validation and user-friendly experience
- **Single File**: Entire website contained in one HTML file for easy deployment

## Deployment

This website is designed for static hosting on GitHub + Cloudflare Pages:

1. Push this repository to GitHub
2. Connect your GitHub repo to Cloudflare Pages
3. Deploy with these settings:
   - Build command: (leave empty)
   - Build output directory: `/`
   - Root directory: `/`

That's it! Your site will be live.

## Sections

- **Hero**: Welcome message with tagline and call-to-action
- **About**: Philosophy and meaning of "Sarvodaya"
- **Space**: 6 amenities (bedroom, dining, yogashala, chill-out space, connectivity, rescued animals)
- **Experiences**: 6 activities (yoga, kayaking, swimming, art, nature, meditation)
- **Pricing**: ₹6,000-10,000 per night with inclusions
- **Location**: Why Varkala and what makes this space special
- **Contact**: Inquiry form with email, Instagram, and location details
- **Footer**: Copyright and tagline

## Technology

- Pure HTML5, CSS3, and vanilla JavaScript
- Google Fonts (Lora and Montserrat)
- Hand-drawn SVG illustrations
- No external dependencies or frameworks

## Local Development

Simply open `index.html` in your browser, or run a local server:

```bash
python3 -m http.server 5000
```

Then visit `http://localhost:5000`
